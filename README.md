# Laravel Payments - Single Charge Practice Project

This is a training project for the Udemy course:
**[Laravel Payments - Stripe and Cashier](https://www.udemy.com/course/laravel-payments-stripe-and-cashier/)**

**Note:** This repository only covers the **Single Charge** section of the course.

## Current Project Status

- **Framework:** Fresh Laravel 12 installation.
- **Authentication:** Laravel Breeze scaffolding is installed (`welcome`, `dashboard`, and `profile` routes).
- **Payments:** Stripe and Laravel Cashier integrations have not yet been started.

## About

This project was created for practice and learning purposes to implement single charge payments using Laravel and Stripe.
